# Screenshots Directory

Place your CollabX application screenshots in this directory.

## Required Screenshots

1. **main-editor.png** - Main editor interface showing the connected state
2. **realtime-collab.png** - Multiple tabs/windows showing real-time collaboration
3. **offline-mode.png** - Editor in offline/disconnected state

## How to Add Screenshots

1. Save your screenshots with the names above
2. Place them in this `docs/screenshots/` directory
3. The README.md will automatically reference them

## Screenshot Guidelines

- **Format:** PNG or JPG
- **Size:** Recommended 1200x800px or similar
- **Quality:** Clear, readable text and UI elements
- **Content:** Show the key features of the application
