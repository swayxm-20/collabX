# How to Add Screenshots

## Step 1: Save Your Screenshots

Save your screenshots with these exact names in the `docs/screenshots/` directory:

1. **main-editor.png** - Main editor interface (the one showing "HI, i am swayam")
2. **realtime-collab.png** - Multiple tabs showing real-time collaboration
3. **connection-status.png** - Connection status indicator (optional, can reuse main-editor.png)

## Step 2: Add to Git

```bash
# Navigate to project root
cd C:\Users\SWAYAM\collabX

# Add screenshots
git add docs/screenshots/*.png

# Commit
git commit -m "docs: Add application screenshots"

# Push
git push origin main
```

## Screenshot Requirements

- **Format:** PNG or JPG
- **Recommended size:** 1200x800px or similar
- **Quality:** Clear, readable text
- **File size:** Keep under 1MB each for GitHub
